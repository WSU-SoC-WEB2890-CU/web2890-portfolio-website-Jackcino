Case Study

This project took several days to complete as it went quite in depth into react modules and vite. I spent time developing the functionality of the website including adding menu items to the cart, being able to edit the cart items in the cart menu, and being able to adjust the amount of how much of the item it added. I also added animation effects to the button so that it moved whenever an item was added to the cart. I also added a connection to a real time data database that supplied the information for the menu items, as well as receiving data from the website when a customer filled out an order. There are also fail safes for slow loading times, on slower networks and there is also an error message if the data from the database were ever to have an issue.
